📋 Cara Mendapatkan Data Instagram

Tutorial Video

📱 Langkah 1: Request Data Instagram

  1. 1 Buka Instagram → Pengaturan → Informasi dan Izin Anda → Unduh Informasi Anda
  2. 2 Pilih "Mengunduh atau Mentransfer Informasi" → Akun Instagram → Beberapa Informasi Anda
  3. 3 Pilih "Pengikut dan Mengikuti" → Unduh ke Perangkat → Beberapa Informasi Anda
  4. 4 Ganti "Retang Tanggal= Sepanjang Waktu" → "Format= Html atau Json" → Buat File
  5. 5 Tunggu email dari Instagram 1x24 Jam

📁 Langkah 2: Extract File

  1. 1 Download file ZIP dari Instagram
  2. 2 Extract dan cari folder "connections"
  3. 3 Upload file "followers_1.json" dan "following.json" atau dengan format Html

🔒 Catatan

Data Anda diproses secara lokal di browser dan tidak disimpan di server manapun.
